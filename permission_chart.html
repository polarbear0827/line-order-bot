🚀 2. Provider (推廣者) 起手式

作為系統管理者，你的主要任務是「維護系統安全」並「交接給替代役男」。

  1. 登入最高權限後台

    進入系統網頁後，輸入你的專屬帳號 (預設為 `polarbear268` 或 `Fangyuan`) 及密碼，登入後會在左側選單看到專屬的 「👑 管理者後台」。

  2. 建立替代役男 (Admin) 帳號

    在「管理者後台 > Admin 管理」中，輸入當前負責處理點餐的替代役男姓名。系統會自動生成一組 admin00X 的帳號及隨機的 4 位數密碼。請將這組帳號密碼交給替代役男。

    役男第一次登入時，系統會強制要求他修改密碼。交接或退役時，你可以直接在這裡刪除舊帳號，並建立新帳號給下一位學弟。

  3. 處理消防員忘記密碼

    當學長忘記密碼時，請他找你。在「管理者後台 > 密碼查閱」中，點擊該學長的密碼欄位，即可顯示他的原始密碼。

  4. 解除 IP 封鎖

    系統有防暴力破解機制，登入錯誤 5 次以上會暫時封鎖該網路 IP。如果有人不小心被鎖了，你可以在「IP 封鎖」分頁手動將其解除。

🧑‍💼 3. Admin (替代役男) 日常作業

替代役男負責維護菜單以及處理每天的算錢與結帳。

  1. 初次設定：建立消防員名單

    進入「👥 人員管理」，可以逐筆新增消防員的代號與姓名（例如：代號 1, 姓名 王大明），或者直接上傳 Excel 批次匯入。

    建立人員的同時，系統會自動生成該員的登入帳號（預設為 user + 代號，如 user001）與 4 位數初始密碼，請統一公布給學長們。

  2. 初次設定：建立店家與菜單

    進入「🏪 店家 / 菜單」，點擊「新增店家」。接著點入該店家，使用 上傳菜單照片 (AI 辨識) 功能，系統會自動將照片轉換為文字菜單及價格。請手動微調有錯漏的文字後儲存。

  3. 每日作業：結算與記帳

    當大家在 LINE 點完餐後，進入「💰 每日記帳」。

    • 查看今天有誰點餐，並點擊右上方按鈕將未標價的品項快速填上價格。
    • 點餐送達後，可以在此頁面直接點擊「未付 / 已付」切換付款狀態。
    • 如果有學長幫別人代墊，可以點擊該筆訂單的「無代墊」，選擇是誰幫忙付的錢，系統會自動轉移債務關係。
🧑‍🚒 4. User (消防員) 操作說明

學長們平時只要在 LINE 群組點餐即可。網頁版主要用來查帳。

  1. 在 LINE 群組點餐

    請參考下方的「LINE 指令大全」。最常用的就是 !點 指令。例如午餐想吃排骨飯，就在群組輸入 !點 午餐 排骨飯 即可。

  2. 登入個人入口查帳

    進入網頁版，輸入自己的帳號 (如 user001) 與初始密碼。第一次登入會被要求改密碼。

    登入後會進入專屬的 「🧑‍🚒 個人入口」：

    • 今日點餐：看今天自己點了什麼。
    • 未付款明細：看自己目前總共欠了多少錢、欠誰錢。
    • 歷史消費：可以選取日期區間，看看上個月吃掉多少伙食費。
📱 5. LINE Bot 指令大全

在群組中輸入以下指令即可與機器人互動（注意：驚嘆號需為半形 !）。

!點 [餐別] [品項]

例如：!點 午餐 雞腿便當
系統會自動辨識你是誰，並記錄訂單。

!取消 [餐別]

例如：!取消 午餐
取消你今天該餐別的點餐紀錄。

!吃什麼

列出今天午餐、晚餐等已經決定的店家名稱及菜單連結。

!誰沒點 [餐別]

例如：!誰沒點 午餐
列出群組內還沒有點餐的學長名單。

!查帳

私訊或在群組輸入，機器人會列出你目前所有未付款的欠帳明細與總金額。

!統計 [餐別]

例如：!統計 午餐
顯示目前午餐的點餐總數量（例如：排骨飯 x3, 雞腿飯 x2），方便打電話叫餐。
