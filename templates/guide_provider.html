📋 登入記錄

記錄所有帳號每一次的登入嘗試，包含成功與失敗：

📋 登入記錄
時間帳號角色IP結果
04/27 14:30 user021 User 192.168.1.5 成功
04/27 14:28 admin001 Admin 192.168.1.3 成功
04/27 13:55 （未知） Admin 203.72.xx.xx 失敗
💡 如果看到大量來自不同 IP 的失敗記錄，表示有人嘗試暴力破解，系統會自動封鎖。
