🔐 如何登入？

打開後台網址，選擇「User」角色，輸入你的帳號和密碼即可登入。

🍱 點餐系統
👑
Provider
🧑‍💼
Admin
🧑‍🚒
User
帳號格式：user + 你的三位數代號
例如：代號 1 → user001，代號 21 → user021

初始密碼：管理員給你的 4 位數字（不知道的話問替代役男）
💡 第一次登入後建議立即去「個人設定」改成自己記得的密碼（4–8 位，數字或英數皆可）

📱 登入後可以做什麼？

🍱
今日點餐
今天點了什麼餐、哪家店、金額，一目了然
💸
目前欠款
目前還有多少錢沒付、欠誰的，清楚列出
📅
歷史消費
選日期範圍，查這段時間吃了多少錢、每餐點了什麼
🔐
改密碼
設定自己好記的密碼（4–8 位，數字或英數混合）

🔐 改密碼

到右上角「個人設定」→「改密碼」，輸入舊密碼和新密碼即可：

🔐 修改密碼
目前密碼
••••
新密碼（4–8 位）

確認修改
💡 密碼 4–8 位，純數字或英數混合都可以，設一個自己好記的就行。

⚠️ 注意事項

  • User 帳號只能看到自己的訂單和帳單，無法看到其他人的資料
  • 金額由替代役男輸入，如有問題請告知
  • 忘記密碼請找替代役男（Admin）重設，或聯絡系統管理者
  • 帳號格式固定為 user + 三位數代號，不可自行修改
